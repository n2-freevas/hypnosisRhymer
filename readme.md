# Hello!
hypnosisRhymer.jsは、ブラウザでライムを踏みたいRapperのために制作されたクールなライブラリです。ブラウザがあれば誰でも操作でき、お手持ちのWebサイトにも簡単に導入できます。自分のリリックとアニメーションを定義して、オリジナルのPVを作ることもできます。<br>
